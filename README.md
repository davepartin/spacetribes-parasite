# Space Tribes: Parasite

An asynchronous **3–6 player** dice-battle (Dave, Brian, Joel, Chris, plus Sam & Riley when you need extra seats).

**v4.3:** 10 rounds — Training → Infection (you + the monster) → Boss. Table size picks who plays; host seats are random (max 2 each). Parasite + boss HP scale with player count. Human mod die, Scar die, exploding attack + specialty die with plain labels. Tap dotted words for tips. Round **Log**. Sticky standings. Live Energy countdown. 1 free reroll every round.

The entire game is one file: **`index.html`**. No build step.

- **[SETUP.md](SETUP.md)** — Firebase + GitHub Pages  
- **[RULES.md](RULES.md)** — full crew manual  

Open the hosted page on your phone, tap **New game** to set the table (3–6), then pick your character. Your phone remembers you.
