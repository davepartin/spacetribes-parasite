# Space Tribes: Parasite

An asynchronous **3–6 player** dice-battle. Type everyone’s name at the table — no fixed character list.

**v4.3.1:** 10 rounds — Training → Infection (you + the monster) → Boss. Pick 3–6, type names, claim your seat. Host seats are random (max 2 each). Parasite + boss HP scale with player count. Human mod die, Scar die, exploding attack + specialty die with plain labels. Tap dotted words for tips. Round **Log**. Sticky standings. Live Energy countdown. 1 free reroll every round.

The entire game is one file: **`index.html`**. No build step.

- **[SETUP.md](SETUP.md)** — Firebase + GitHub Pages  
- **[RULES.md](RULES.md)** — full crew manual  

Open the hosted page on your phone, tap **New game**, set 3–6 players, type the names, then Join your seat. Your phone remembers you.
