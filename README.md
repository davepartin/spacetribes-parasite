# Space Tribes: Parasite

An asynchronous 4-player dice-battle game for Dave, Brian, Joel & Chris. A hostile alien parasite possesses one crew member each round — humans forge custom die faces with **Energy**, the parasite grows **attack + modifier dice**, scores stay secret until the final reveal after round 8.

The entire game is one file: **`index.html`**. No build step, no server code.

- **[SETUP.md](SETUP.md)** — 5-minute walkthrough: free Firebase database + GitHub Pages hosting
- **[RULES.md](RULES.md)** — Energy, face forging, parasite modifiers, scoring

Open the hosted page on your phone, tap your character once, and your phone remembers you. Play a round a day or blitz a whole game in an evening.

**Schema v2:** if you still have an old game in Firebase, the app will ask you to start a new one — old token-based saves are not compatible.
