# Space Tribes: Parasite

An asynchronous 4-player dice-battle for Dave, Brian, Joel & Chris.

**v4.2:** 10 rounds — Training → Infection (you + the monster) → Boss. Human mod die (2-2-2), Scar die for heavy growth, exploding attack dice + specialty die with plain effect labels. Tap dotted words for tips. Round **Log**. Sticky standings. Live Energy countdown. 1 free reroll every round.

The entire game is one file: **`index.html`**. No build step.

- **[SETUP.md](SETUP.md)** — Firebase + GitHub Pages  
- **[RULES.md](RULES.md)** — full crew manual  

Open the hosted page on your phone, tap your character once, and your phone remembers you.

Start a **New game** after this update so everyone is on the same build.
